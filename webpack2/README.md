
ag-grid-angular-examples - webpack 2
==============================

Example of using ag-Grid with Angular, Webpack 2 and TypeScript

Building
========

Install Dependencies:

- `npm install`

Dev Build
=========

To build & run:

- `npm start`

Prod Build
=========

To build:

- `npm run build`

e2e Tests
=========

Run Dev Server:

- `npm start`

In a new Terminal, run the tests:

- `npm run e2e`
